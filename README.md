# TODOAPP_cli
Sample todoapp connected on the JSMB's todoapp API (public repo coming soon)

## Build
```
cargo build --release
```

## Run
```
cargo run --release
```

## Main libraries

*[reqwest](https://github.com/seanmonstar/reqwest): API interface
*[Cursive](https://github.com/gyscos/cursive): TUI interface
